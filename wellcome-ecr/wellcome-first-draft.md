Role models; “fit for purpose”; “good enough”

My research plans are organised around three steps of evaluating policy-facing modelling work: credible outputs, informing relevant outcomes, using legitimate processes.

WP1. **Credible** modelling & boundaries of predictability. 

The first work package explores model evaluation from the perspective of self-contained model quality. This aims to identify the boundaries of predictability in outbreak modelling, helping us to evaluate appropriately the performance that we can expect for any given model. This can be achieved by iterating between two extremes. 

From an applied perspective, we can consider the actualised performance across a diversity of forecasting models. By decomposing the error structure of forecasts from many different models, we can start to identify patterns in which models perform well according to various criteria, for example, consistency; accuracy at critical epidemiological dynamics, e.g the timing of the peak or the start of exponential growth. Decomposing elements of accuracy in this way may also prompt the development of scoring metrics that differentially weight accuracy according to epidemiological relevance, providing new metrics to support model development.

From a theoretical perspective, we can consider the “lowest common denominator” in the construction of a credible model. This varies between model types, with several broad categories from strictly mechanistic, to purely statistical. This approach involves expert elicitation to identify the fundamental theory underlying each archetypal model. This could be operationalised into a suite of readily available baseline models which modellers may adopt in independent evaluations. At the same time, this process can be used to identify factors that fundamentally limit forecast accuracy, such as unknown change in transmission patterns. Identifying these factors mean they can be adjusted for in evaluation, creating a fair playing field from which to evaluate individual model performance.

Characterising the limits of predictability, among model types will help establish an appropriate baseline from which to evaluate whether any specific model is an improvement, thus supporting where we invest in model development.
